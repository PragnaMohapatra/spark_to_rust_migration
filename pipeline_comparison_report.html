Analysis: Rust outperforms Spark on every latency measure. The biggest win is in Delta Lake writes (7.4× faster on 500K batches), which is the dominant cost in both pipelines. Transform speedup is 2.7×. The wall-clock difference is smaller (1.15×) because Rust started earlier and finished sooner, but spent more idle time between smaller tail batches waiting on Kafka. For large continuous workloads the per-batch advantage compounds further.
